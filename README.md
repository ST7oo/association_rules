# Association rules

To run:

 - Install [nodejs](https://nodejs.org/)
 - Go to the project folder
 - Run `npm install`
 - Run `npm start` (it will open a browser tab, and watch for changes)

To use:

 - Click Reset
 - Add a row (observation) separated by `,`
 - Click Calculate
	 - Change Support threshold
